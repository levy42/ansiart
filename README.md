# ansiart
### Tool for generating ANSI pictures from images.

###### There are a cmd tool and a simple web app.
CMD usage:
```sh
$ python ansiart <path-to-image> [size] [palette] [inverse (True|False)]
```

To run web application:
  ```sh
$ python web.py
```
